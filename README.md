# bootstrap
## Objective
The objective of this project was to explore Bootstrap 5 and design responsive web pages by combining different UI components.
## Steps I Followed
- I studied Bootstrap 5 documentation and example templates
- I selected components like navbar, hero section, cards, and forms
- I created three pages: Home, About, and Contact
- I customized styles and layout instead of copying full templates
- I ensured consistency in colors, spacing, and responsiveness
## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- GitHub (for hosting)
## Challenges Faced
- Understanding how to combine different Bootstrap components
- Maintaining consistent design across pages
- Adjusting spacing and layout properly
## How I Solved Them
- Referred to Bootstrap documentation
- Used utility classes like spacing (mt, mb, py)
- Modified components step by step
## Learning Outcome
- Learned how to use Bootstrap 5 effectively
- Improved UI/UX design skills
- Understood responsive design principles
- Learned how to deploy a website using GitHub Pages
## Tools Used
- Bootstrap documentation
- ChatGPT (for guidance and debugging)
